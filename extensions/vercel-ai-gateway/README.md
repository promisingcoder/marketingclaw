# Vercel AI Gateway MarketingClaw provider

Official MarketingClaw provider plugin for Vercel AI Gateway.

## Install

```sh
marketingclaw plugins install @marketingclaw/vercel-ai-gateway-provider
```

## Docs

See `docs/providers/vercel-ai-gateway.md` in the MarketingClaw repository, or the published docs at `https://docs.marketingclaw.ai/providers/vercel-ai-gateway`.
