# Tencent Cloud MarketingClaw provider

Official MarketingClaw provider plugin for Tencent Cloud.

## Install

```sh
marketingclaw plugins install @marketingclaw/tencent-provider
```

## Docs

See `docs/providers/tencent.md` in the MarketingClaw repository, or the published docs at `https://docs.marketingclaw.ai/providers/tencent`.
