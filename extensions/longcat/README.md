# MarketingClaw LongCat Provider

Official MarketingClaw provider plugin for LongCat-2.0 through LongCat's hosted
OpenAI-compatible API.

Install from MarketingClaw:

```bash
marketingclaw plugins install @marketingclaw/longcat-provider
marketingclaw gateway restart
```

Configure `LONGCAT_API_KEY`, then select `longcat/LongCat-2.0`.
