# MarketingClaw Discord

Official MarketingClaw channel plugin for Discord servers, channels, DMs, slash commands, and app events.

Install from MarketingClaw:

```bash
marketingclaw plugin add @marketingclaw/discord
```

Configure a Discord bot token and the channels or servers MarketingClaw should handle. The plugin lets MarketingClaw agents receive Discord messages and respond through the configured Discord app.
