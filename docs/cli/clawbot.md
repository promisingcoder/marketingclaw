# `marketingclaw clawbot`

Legacy alias namespace kept for backward compatibility. It registers the same QR command as the top-level CLI, so `marketingclaw clawbot qr` accepts every [`marketingclaw qr`](/cli/qr) flag.

## Migration

Prefer the modern top-level command:

- `marketingclaw clawbot qr` -> `marketingclaw qr`
