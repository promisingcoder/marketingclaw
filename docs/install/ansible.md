<Warning>
This page inherited an Ansible playbook reference from upstream OpenClaw
([openclaw/openclaw-ansible](https://github.com/openclaw/openclaw-ansible)).
Running that playbook deploys the **upstream OpenClaw engine, not
MarketingClaw** — this fork has no Ansible playbook of its own yet.

For a real MarketingClaw deployment, install from source instead: see the
[Marketing quick start](/start/marketing-quickstart).
</Warning>
